# Fitness Tracker

A full-stack gym and nutrition tracking app
Building this as I don't find any other apps to align with my goals without a high monthly cost.

Happy for others to use, Health shouldn't have a pay wall.

## Tech Stack

- **Backend:** C# ASP.NET Core 8 Web API
- **Database:** SQL Server (via Docker locally, Azure SQL in production)
- **Frontend:** React + Recharts *(coming)*
- **Hosting:** Azure App Service *(coming)*
- **CI/CD:** GitHub Actions *(coming)*
